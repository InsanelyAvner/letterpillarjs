LetterPillar Modifications by Avner

Tired of doing LetterPillar? Want to have some fun? You've come to the right place!

These changes are temporary, any changes made will not be saved

Step #1:

Log in to Letterpillar and get into the system.

Press CTRL+SHIFT+J together on your keyboard. A console will pop up. Remember to follow the steps carefully :)

Step #2:

Select/highlight the code below. Afterwards, copy the code by pressing CTRL+C together.

Alternatively, you can press Copy

Step #3:

Once you've copied the code, navigate to the Letterpillar console.

Paste the code into the console by pressing CTRL+V together.

Finally,

Hit ENTER. Close the console by pressing the tiny x located on the top-right corner.

You're good to go! You should be seeing some changes on website.
